## 🚀 快速命令参考

```bash
# 环境准备
./setup.ps1  # Windows
./setup.sh   # Linux/Mac

# 基本流程
python RustFuzz.py preprocess -L <library>    # 预处理
python RustFuzz.py generate -L <library>      # 生成
python RustFuzz.py fuzz -L <library>          # 运行
python RustFuzz.py analyze -L <library>       # 分析
```
